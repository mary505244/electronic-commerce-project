# 1 安裝redux相關依賴

* 命令：

```shell
yarn add redux redux-thunk react-redux redux-devtools-extension
```

# 2 安裝axios

* 命令：

```shell
yarn add axios
```

# 3 安裝ngprogress

```shell
yarn add nprogress
```

# 4 安裝screenFull

```shell
yarn add screenfull
```

# 5 安裝dayjsC

```shell
yarn add dayjs
```

# 6 安裝jsonp（本項目中沒有使用到）

```shell
yarn add jsonp
```
